내 Google Sheets 연결

본인의 Google Apps Script 웹앱 URL을 입력하면 영수증 데이터가 내 Google Sheets에 저장됩니다. 비워두면 기본 저장소(localStorage)만 사용됩니다.

주의사항
  • 학교/회사 계정(Google Workspace)으로 만든 Google Sheets는 도메인 보안 정책으로 인해 연결이 제한될 수 있습니다.
  • 개인 Gmail 계정으로 새 Google Sheets를 만들어 사용해주세요.
  • 배포 시 "실행: 나" / "액세스 권한: 모든 사용자"로 설정했는지 확인해주세요.
Google Sheets 준비 방법
Google Sheets에 receipts 탭을 만들고 1행에 아래 컬럼을 순서대로 입력하세요.
id date store_name amount category payment_method memo created_at items author
내 Google Sheets 설정 방법 (단계별 안내)
1
Google Sheets 새로 만들기
sheets.google.com에서 새 스프레드시트를 만들고, 하단 탭 이름을 receipts로 변경합니다. 1행에 아래 컬럼을 순서대로 입력하세요.
id date store_name amount category payment_method memo created_at items author
2
Apps Script 열기
상단 메뉴 확장 프로그램 > Apps Script를 클릭합니다. 새 탭에 스크립트 편집기가 열립니다.
3
아래 코드를 붙여넣기
편집기의 기존 내용을 모두 지우고 아래 코드를 붙여넣습니다.
💡 Sheets 1행 헤더를 기준으로 자동 매핑됩니다. 컬럼을 추가해도 코드 수정 없이 자동 인식됩니다.
function doGet(e) { var p = e.parameter; var ss = SpreadsheetApp.getActiveSpreadsheet(); var sheet = ss.getSheetByName(p.table); var result; if (!sheet) { result = { status: 'error', message: 'Sheet not found: ' + p.table }; } else if (p.action === 'insert') { var data = JSON.parse(p.data); var headers = sheet.getRange(1, 1, 1, sheet.getLastColumn()).getValues()[0]; sheet.appendRow(headers.map(function(h) { return data[h] !== undefined ? data[h] : ''; })); result = { status: 'ok' }; } else if (p.action === 'read') { var values = sheet.getDataRange().getValues(); var headers = values[0]; result = { status: 'ok', data: values.slice(1).map(function(row) { var obj = {}; headers.forEach(function(h, i) { obj[h] = row[i]; }); return obj; }) }; } else if (p.action === 'delete') { var values = sheet.getDataRange().getValues(); var targetId = p.id; var deleted = false; for (var i = 1; i < values.length; i++) { if (String(values[i][0]) === String(targetId)) { sheet.deleteRow(i + 1); deleted = true; break; } } result = deleted ? { status: 'ok', message: 'deleted' } : { status: 'error', message: 'id not found' }; } else { result = { status: 'error', message: 'unknown action' }; } return ContentService .createTextOutput(p.callback + '(' + JSON.stringify(result) + ')') .setMimeType(ContentService.MimeType.JAVASCRIPT); }
⚠ 코드 수정 후 반드시 새 버전으로 재배포해야 합니다.
배포 > 배포 관리 > 연필 아이콘 클릭 > 버전: 새 버전 선택 > 배포.
기존 배포를 그냥 저장하면 변경사항이 반영되지 않습니다.
4
저장 및 웹앱으로 배포
Ctrl+S로 저장 후 우측 상단 배포 > 새 배포를 클릭합니다.
유형: 웹 앱 · 실행: 나(본인 Gmail) · 액세스: 모든 사용자(익명 포함)로 설정 후 배포. 권한 승인 팝업이 뜨면 허용합니다.
5
URL 복사 후 입력
배포 완료 후 표시된 웹앱 URL(https://script.google.com/macros/s/.../exec)을 복사하여 위 입력창에 붙여넣고 저장 후 연결 테스트를 눌러 확인합니다.
